SSH Remote Support

Click Start to create a temporary support session. You will be asked for an administrator password to enable Remote Login and apply temporary SSH restrictions. After it starts, a new page will open showing the SSH command to send to support.

Tip: If your browser asks for confirmation to open “SSH Tool”, click Open.
Safety notes (recommended defaults):
  • By default, SSH is exposed only through the bore.pub tunnel (not your LAN).
  • If a support public key is bundled, password login is disabled. Otherwise a temporary user + random password is used.
  • Session auto-stops at the expiry time and restores previous settings.
